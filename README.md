## AI Tool Utilized

**ChatGPT** – Used for generating smart schedules, reminders, and productivity suggestions based on user prompts.

---

## Use Case Description

### **Scenario:**

A college student wants an AI-powered assistant to manage academic work, coding practice, project activities, and personal tasks efficiently.

The assistant should:

* Generate a daily task schedule
* Organize activities based on priority
* Provide reminders and productivity tips

### **Prompt Context:**

“You are a smart productivity assistant. Create a balanced daily task schedule for a college student with study sessions, coding practice, project work, and relaxation time.”

---

## Problem Statement

How can prompt engineering be used to transform ChatGPT into a personalized task management assistant capable of organizing schedules and improving daily productivity?

---

## Procedure

1. Identify daily productivity requirements.
2. Design prompts for task scheduling and reminders.
3. Generate AI-based responses using ChatGPT.
4. Refine prompts for better personalization.
5. Include wellness and productivity suggestions.

---

## Objective Output

A personalized AI productivity assistant capable of:

* Creating smart daily schedules
* Managing task priorities
* Tracking completed activities
* Providing motivation and reminders

## Algorithm Blueprint

The AI assistant generates a balanced productivity schedule based on user priorities and available time.

### **Input Parameters**

* **User Goal:** Improve daily productivity
* **Activities:** Study, Coding, Projects, Exercise
* **Preferences:**

  * 5-hour productive schedule
  * Break reminders enabled
  * Motivation notifications included

### **Expected Output**

* Personalized daily routine
* Task priority management
* Progress tracking summary

---

## Step-by-Step Algorithm Design

1. **Collect User Activities** – Gather daily tasks and priorities.
2. **Analyze Importance** – Identify high-priority activities.
3. **Generate Schedule** – Create a balanced timetable.
4. **Insert Breaks** – Add wellness and relaxation intervals.
5. **Track Progress** – Monitor completed and pending tasks.
6. **Provide Motivation** – Display encouraging productivity tips.
7. **Update Schedule** – Modify the next plan based on progress.
